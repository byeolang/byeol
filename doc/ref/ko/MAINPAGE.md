# 안녕하세요.

`byeol` c++ 인터프리터 프로젝트의 레퍼런스 문서입니다.
c++ 소스코드 파일에 대한 레퍼런스 문서와 코드 이해를 돕기 위한 가이드 문서로
구성되어 있습니다.

클래스 레퍼런스는 왼쪽에서 [모듈](topics.html)별로 검색하거나, [클래스
색인](classes.html)에서 찾거나 <b>작은 검색창</b>을 이용하면 됩니다.

그러나 이 프로젝트에 코드를 읽기 위해 <b>처음으로</b> 마음을 먹으셨다면 <b>일단 코드를 보기 전에 가이드 문서</b>를
순서대로 읽어보시길 권장합니다. 가이드 문서는 왼쪽 메뉴에, 읽는 순서를 고려하여 위에서부터 배치되어 있습니다.

c++에 숙련된 개발자를 대상으로 작성되었으며 byeol 언어 문법을 다 알고
있다는 전제로 작성되었습니다. byeol 언어 문법에 대해 잘 모른다면 [언어 가이드](https://byeol.io/guide/)를 먼저 읽어보시길 권장합니다.
